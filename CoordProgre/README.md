# DESCRIPCIÓN
Este codigo en Python crea rotulados de progresivas a partir de ruta en formato kml
## REQUERIMIENTOS
+ PYTHON VERSION >= 3.11.8
+ LIBRERIA PYPROJ
+ LIBRERIA LXML
